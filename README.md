# Spotify Data Analysis Dashboard
## Project Overview

This project presents an interactive Spotify Data Analysis Dashboard built using Power BI. The dashboard analyzes Spotify music data to understand trends in songs, artists, and music popularity.

The goal of this project is to transform raw Spotify data into meaningful visual insights that help understand music trends, artist popularity, and song performance.

## Objectives

The main objectives of this project are:

• Analyze trends in popular Spotify songs

• Identify top artists and most streamed tracks

• Understand distribution of songs across different years

• Study audio features and music characteristics

• Provide interactive visual insights using Power BI

## Dataset Description

The dataset used in this project contains information about Spotify songs such as:

• Track Name

• Artist Name

• Album Name

• Release Year

• Popularity Score

• Danceability

• Energy

• Tempo

• Loudness

• Speechiness

• Acousticness

• Instrumentalness

• Valence

These features help analyze the musical characteristics of songs and their popularity.

## Tools and Technologies Used

The following tools were used in this project:

• Power BI

• Data Modeling

• Data Visualization

• DAX (Data Analysis Expressions)

## Dashboard Features

The Spotify dashboard includes the following visualizations:

• Total Songs Overview

• Artist Popularity Analysis

• Song Popularity Distribution

• Year Wise Song Distribution

• Audio Feature Analysis

• Top Artists and Tracks

These visualizations allow users to explore Spotify data interactively.

## Key Insights

Some insights obtained from the dashboard include:

• Identification of the most popular artists

• Distribution of songs across different years

• Audio characteristics of popular songs

• Trends in music popularity and streaming performance
